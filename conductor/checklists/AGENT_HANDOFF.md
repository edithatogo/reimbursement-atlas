# Agent handoff checklist

Before starting work:

- [ ] Read `conductor/INDEX.md`.
- [ ] Read `conductor/context/CANONICAL_CONTEXT.md`.
- [ ] Check `conductor/DECISION_LOG.md`.
- [ ] Check relevant ADRs.
- [ ] Validate seed data.
- [ ] Check licence notes before using any external source.
- [ ] Confirm whether the work changes source registry, analysis catalogue or ontology registry.
- [ ] Add tests before or with implementation.
- [ ] Update docs if architecture or policy assumptions change.

Before finishing work:

- [ ] Run formatting/linting/tests relevant to the change.
- [ ] Update Conductor context if a decision was made.
- [ ] Add an ADR for architecture changes.
- [ ] Add provenance for new source data.
- [ ] Keep restricted data out of Git.
