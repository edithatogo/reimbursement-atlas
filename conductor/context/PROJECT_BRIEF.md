# Project brief

## Name

Reimbursement Atlas Conductor

## Mission

Build a reproducible, policy-relevant comparative atlas of public reimbursement settings across US CMS, Australian MBS/PBS and other public reimbursement systems.

## Primary users

- Health economists.
- Clinician-researchers.
- HTA and reimbursement policy analysts.
- Market-access analysts.
- Medical administrators.
- AI agents assisting with source mapping and analysis.

## Near-term objective

Mature the design before implementation by creating:

- requirements;
- architecture diagrams;
- source registry;
- analysis catalogue;
- ontology strategy;
- seed data;
- CI/security plan;
- dashboard scaffold;
- GitHub project plan.

## First research focus

Genomics/pathology, specialist services, high-cost medicines and hospital episode bundling.

## Non-goals now

- patient-level data analysis;
- production API;
- public release of restricted ontology data;
- direct claims of cross-country price equivalence.
