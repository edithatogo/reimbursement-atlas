# ADR 0030: Research linkages as project-planning evidence

## Status

Accepted.

## Context

The atlas has Conductor tracks, research questions, dataset candidates, mapping resources and output plans. Without explicit linkage tables, it is difficult to know whether a research question is actually implementable.

## Decision

Generate a research linkage matrix that connects each research question to registered sources, dataset candidates, mapping resources and output artefacts.

## Consequences

The roadmap, OSF plan, dashboard and generated GitHub issue backlog can all show missing prerequisites before research outputs are claimed as ready.
