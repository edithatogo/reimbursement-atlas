# Research protocols

Each protocol defines the research question, eligible sources, mapping standard, price concept, analysis plan, sensitivity analyses, reviewer gates and publication outputs before results are interpreted.

Protocols are designed for OSF registration and should be treated as the analysis source of truth.

Before preregistration, complete the [research protocol review checklist](../docs/RESEARCH_PROTOCOL_REVIEW_CHECKLIST.md)
for each protocol and record the decision in Conductor and the linked GitHub issue.
