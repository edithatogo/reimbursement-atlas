# Hugging Face deployment

## Dataset

A future dataset can contain:

- source registry metadata;
- analysis catalogue metadata;
- ontology registry metadata;
- graph nodes and edges;
- derived permissive analysis outputs.

It must not contain restricted ontology dumps, proprietary descriptors, raw scraped content with unclear rights, or confidential prices.

## Space

The dashboard should be deployed as a static Astro Space first. Later API-backed modes can be added if needed.
